Markdown 是一种书写结构化文档的轻量标记语言，它定制了一种原文本书写格式，利用这种格式书写的文档内容结构清晰，查看方便，书写也很简单，没有多少标签符号，不像 HTML 那样需要写大量标签标记，也不需要用额外的 IDE 来编辑，随便一个文本编辑器就可开工，可以说是一种很轻巧、灵活、随性的内容记录标记语言。
